# SkyGrid
![alt text](https://apex-misc.s3.amazonaws.com/Images/Skygrid_Thumbnail.png)
This gamemode is a modern take on an old school classic. Jump around in the crazy world of blocks and see how far you can go, Many options and settings are configurable.

https://apexminecrafthosting.com/skygrid-server/
